# Project-Management-Gantt-Chart-Resource-Planner


A professional Excel-based Project Management Dashboard designed for project tracking, resource planning, task monitoring, and data analysis.


Features


- Raw and Cleaned Data Management
- Interactive Dashboard Navigation using Hyperlinks
- Gantt Chart for Project Timeline Tracking
- Resource Planner with Automated Calculations
- Task Tracker and Progress Monitoring
- Data Cleaning and Validation Process
- Excel Formula Automation (SUM, AVERAGE, etc.)
- Professional Charts and Visualizations
- Validation and Quality Check Reports


Technologies Used


- Microsoft Excel
- Excel Formulas
- Data Cleaning Techniques
- Dashboard Visualization
- Project Management Concepts

Project Highlights

- Maintains data integrity with validated cleaned datasets
- Contains 150 project records for analysis
- Includes professional formatting and structured reporting
- Designed to simulate real-world business/project tracking systems

Purpose

This project demonstrates the use of Excel for:

- Project Management
- Data Analysis
- Dashboard Development
- Resource Allocation
- Task Monitoring
- Reporting and Visualization

Author

Developed as an academic/project management dashboard project using Microsoft Excel.
